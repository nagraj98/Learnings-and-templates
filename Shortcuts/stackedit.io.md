# Stackedit.io
An online tool to easily write and edit markdown.
Link - https://stackedit.io/


##Shortcuts :

Code - Ctrl+Shft+K
Link - Ctrl+Shft+L
Image - Ctrl+Shft+G
BlockQuote - Ctrl+Shft+Q
Checklist - Ctrl+Shft+C
Orderedlist - Ctrl+Shft+O
Unorderedlist - Ctrl+Shft+U
Strikethrough - Ctrl+Shft+S
Heading - Ctrl+Shft+H
Italics - Ctrl+Shft+I
Bold -Ctrl+Shft+B
